# Tourist Attractions Data

JSON datasets for tourist attractions across West Coast cities.

## Major Cities (7 cities, ~876 attractions)

- Reno, NV
- Seattle, WA
- Portland, OR
- Mammoth Lakes, CA
- New Orleans, LA
- Santa Barbara, CA

## Schema
Each attraction includes: name, category, description, address, city, region, hours, days_open, admission, website, phone, image_url, highlights, tips, nearby.
